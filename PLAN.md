# App Pages (Wireframe in words)

1. Login / Sign Up page - email, password fields, button
2. Dashboard - list of past code reviews, a button "New Review"
3. New Review page - textbox to paste code OR upload a file, button "Analyze"
4. Results page - shows list of issues found, each with severity (Low/Medium/High), explanation, suggested fix
5. Profile page - shows user's name/email, logout button
